
# Simple Python Calculator

A basic calculator coded in Python 3.1 utilizing the `tkinter` library as practice for programming logic and GUI building.

# Functionalities

* Basic mathematical operations: addition (`+`), subtraction (`-`), multiplication (`×`) and division (`÷`);
* Other functions: square root (`√`), percentage (`%`) and decimal point support (`.`);
* Basic controls: positive/negative sign inversion (`+/-`) and All Clear button (`AC`);
* Interface: simple interface inspired by classic smartphone's calculators

# How to run this project

### Prerequisites
[Python](https://www.python.org/downloads/) is needed to be installed on your computer. The `tkinter` library comes built-in with the standard Python installation.

### Steps

#### 1. Clone this repository in your terminal:

```bash
git clone https://github.com/guikkambara/simple-calculator-python.git
```

#### 2. Navigate to the project folder:

```bash 
cd simple-calculator-python
```

#### 3. Run the main file:

```bash
python calculator.py
```
